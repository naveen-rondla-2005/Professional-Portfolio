# Naveen Rondla - AI Software Engineer Portfolio

A premium, interactive, and high-contrast developer portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Featuring a dual-theme "Neural-Glow" aesthetic with 3D elements and a terminal-inspired bio.

## 🚀 Key Features

- **Dual-Theme "Neural-Glow"**: High-contrast Light Mode and vibrant "Future-Dark" Mode.
- **Interactive 3D Elements**: Custom Spline components and anti-gravity particle systems.
- **TerminalBio**: Simulated terminal environment for technical profile presentation.
- **Snake Water Gun Game**: A built-in interactive game to showcase logic and UI polish.
- **Professional Skill Taxonomy**: Organized AIML, Core CS, and Tooling sections.
- **Responsive & Accessible**: Optimized for all devices with a focus on readability and professionalism.

## 📄 Live Demo

Check out the live portfolio: [**naveen-rondla-portfolio.vercel.app**](https://naveen-rondla-portfolio.vercel.app)

## 🛠️ Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS 4, CSS Variables
- **Animations**: Framer Motion, Three.js
- **Interaction**: Lucide React, React Calendly
- **Deployment**: Vercel

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/naveen-rondla-2005/Professional-Portfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
